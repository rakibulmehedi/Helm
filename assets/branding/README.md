# Helm App Icon

Canonical direction: `VISR-029` in
`docs/ux/extracted/08_visual_identity_refinements.md`.

- Full-bleed source: `helm_app_icon_master.svg`
- Rounded-canvas source: `helm_app_icon_rounded.svg`
- Background: Helm interactive deep teal `#255E5B`
- Mark: warm white `#FAFAF6`
- Motif: ledger rail, settlement marker, and compact BDT grounding mark
- Constraints: no gradient, shadow, generic wallet, coin, chart, or graph

Full-bleed art serves iOS, Android adaptive icons, and maskable web icons.
Rounded-canvas art serves legacy Android, macOS, standard web icons, and
favicon. Android also uses an adaptive icon foreground in
`android/app/src/main/res/drawable/`.
